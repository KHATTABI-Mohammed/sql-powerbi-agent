# Table: products

Catalogue des produits vendus.

| Colonne     | Type    | Description                                |
|-------------|---------|---------------------------------------------|
| product_id  | INTEGER | Identifiant unique du produit (clé primaire)|
| name        | TEXT    | Nom du produit                              |
| category    | TEXT    | Catégorie (ex: "Logiciel", "Matériel")      |
| unit_price  | REAL    | Prix unitaire en euros                      |
